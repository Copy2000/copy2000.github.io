JNA的应用

在Windows中的Preferences中选择Build Path下的User Library

添加之前在github下载的三个文件。

之后在文件夹，右键点击Build Path下面的Add Library选中我们添加的JNA。

按住ctrl+click你想看到的函数源码，之后点击attach...的button，添加之前在github下载的zip文件，就可以看到源码了。
