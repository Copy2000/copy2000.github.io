hexo的metery主题配置

修改导航栏颜色

打开 themes/matery/source/css/matery.css 文件，找到bg-color这个属性，即为导航栏和底部的颜色

.bg-color {
    background-image: linear-gradient(to right, #8cbfb8 0%, #128c6f 100%);
}

关于主页图片被颜色覆盖

themes\hexo-theme-matery\source\css中一个叫matery.css文件的595行代码造成的，如下：

.bg-cover:before,
.bg-cover:after {
    position: absolute;
    z-index: 1;
    width: 100%;
    height: 100%;
    display: block;
    left: 0;
    top: 0;
    content: ""; 
}

只要将content属性注释掉即可，如下：

.bg-cover:before,
.bg-cover:after {
    position: absolute;
    z-index: 1;
    width: 100%;
    height: 100%;
    display: block;
    left: 0;
    top: 0;
    /* content: ""; */ 
}
